# Where to go from here

You ran a program. Pick a longer path.

## Four onboarding paths

| Path | When |
|---|---|
| **[The Aperio Grimoire](../../grimoire/book/index.html)** | If the spell-cast framing draws you. Magical register; four-moment narrative arc. |
| **[The Aperio Programming Language](../../book/book/index.html)** | If you want a systems-language tutorial. Substrate-up; layered chapters from `hello, locus` through a multi-binary capstone. |
| **[The Aperio Reference](../../reference/book/index.html)** | If you want exhaustive answers to "what exactly does this construct mean." Formal grammar + semantics + glossary. |
| **[The Aperio Standard Library](../../std/book/index.html)** | If you want library-level batteries (I/O, HTTP, text, test framework). Phases 1–5 of the v1.x roadmap. |

The grimoire and the technical book teach the same material in
different registers. Pick the one whose tone reads more clearly to
you; either works.

## Or: read the example ladder

`examples/` in the repository is a 50+ project ladder from
`hello-world` to a multi-binary `trellis-pair` capstone. Every
project has a README walk-through and a runnable `main.ap`. Reading
the ladder front-to-back is itself a tutorial.
