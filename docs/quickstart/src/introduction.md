# The Aperio Quickstart

Five minutes. Install the toolchain, run a one-locus program, learn
where to read next.

This is the shortest of the four onboarding paths. If you want
narrative, the [grimoire](../../grimoire/book/index.html) is the
spell-cast tour and *[The Aperio Programming
Language](../../book/book/index.html)* is the substrate-up tutorial.
If you want the formal grammar, go straight to the
[reference](../../reference/book/index.html).

This page exists for readers who want to verify Aperio works on their
machine before committing to anything longer.
