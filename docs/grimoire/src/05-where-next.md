# Where to go next

You hold the meta-spell. The wand is in your hand. From this point
every Aperio program ever written — and every one not yet written —
is a spell whose shape you already know.

Some specific places to reach next:

- **[The Aperio Programming Language](../../book/book/index.html)** —
  the same material in systems-language register. Substrate-up,
  layered tutorial. Read this when you want the precise mechanics
  behind each rune you've learned.
- **[The Aperio Reference](../../reference/book/index.html)** —
  formal grammar and semantics. The authoritative source for "what
  does this construct *exactly* mean." A grimoire of definitions,
  written without metaphor.
- **[The Aperio Standard Library](../../std/book/index.html)** —
  prebuilt spells: I/O, HTTP, text processing, the test framework.
  (Phases 1–5 of the v1.x roadmap; many libraries are in active
  development.)
- **The example ladder.** `examples/` in the repository is a
  fifty-rung ladder of runnable Aperio programs from `hello-world`
  through a multi-binary `trellis-pair`. Every rung is a spell you
  can cast and modify.

Or: write your own.
